# Cantina On

## Sobre o Projeto

O **Cantina On** é um aplicativo mobile desenvolvido para facilitar a experiência dos alunos da FIAP na compra de lanches e bebidas na cantina da instituição. O app resolve o problema das filas e da falta de praticidade no momento do intervalo, permitindo que o aluno faça pedidos, adicione saldo, visualize produtos e acompanhe seus pedidos de forma digital e rápida.

**Operação FIAP escolhida:** Cantina

A cantina foi escolhida por ser um ponto de grande fluxo e necessidade de otimização no dia a dia dos alunos. A digitalização do processo de compra traz mais agilidade, controle financeiro e comodidade para todos.

### Funcionalidades implementadas
- Visualização de produtos disponíveis na cantina
- Adição de produtos ao carrinho
- Visualização e edição do carrinho
- Adição de saldo à conta do aluno
- Pagamento do pedido
- Visualização de perfil e saldo
- Ajuda e suporte ao usuário

## Como Rodar o Projeto

### Pré-requisitos
- Node.js (recomendado v18+)
- Expo CLI (`npm install -g expo-cli`)
- Expo Go (app para Android/iOS)
- Git

### Passo a passo
1. Clone o repositório:
   ```sh
   git clone <URL_DO_REPOSITORIO>
   ```
2. Acesse a pasta do projeto:
   ```sh
   cd cantina-on
   ```
3. Instale as dependências:
   ```sh
   npm install
   npx expo install expo-router react-native-safe-area-context react-native-screens
   ```
4. Inicie o projeto:
   ```sh
   npx expo start
   ```
5. Escaneie o QR Code com o app Expo Go no seu celular para rodar o app.
